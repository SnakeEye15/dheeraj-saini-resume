## üí° Customization

The design is built entirely with Tailwind CSS utilities and a few custom CSS classes.

* **Colors:** Modify the color variables (e.g., `#58a6ff` for the accent blue) in the main `<style>` block to change the primary theme color.
* **Content:** All text content is in the main HTML file. Update the text fields in the `Profile`, `Skills`, `Experience`, and `Education` sections.
* **Animations:** The entrance animations are controlled by `data-aos="..."` attributes directly on the HTML elements. You can change `fade-right`, `zoom-in`, etc., or adjust the `data-aos-delay` to change the timing.
